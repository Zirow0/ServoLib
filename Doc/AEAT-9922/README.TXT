I.  OPERATING SYSTEM REQUIREMENTS
    Windows 10 64-bit


II. INSTALLATION

    1. Click on the "HEDS-9922 PRGEVB Programming Evaluation Kit.msi" to start the installation.
    2. Click "Next >" buttion.
    3. Choose target directory (point to your desktop) then click "Next >" button.
    4. Click "Next>" button to begin installation.
    5. Click "Close" button once installation complete then restart your computer. 


III. OPERATION FLOW
    1. Connect the Programming kit to Encoder.
    2. Connect the Programming kit to PC USB.
    3. Clink on "Connection" icon.
	"Connection Status: STM32 Kit available -> Connected" message will show at the bottom of GUI.
    4. Refer to application notes for more details.
    5. For different axis configuration, Import config file below using "Import Memory" GUI Form
       a. ON-axis : ON-axis Config.txt
       b. OFF-axis : OFF-axis Config.txt
    6. Select "Configuration Register" and "Calibration Register".

IV. TROUBLESHOOTING
